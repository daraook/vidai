# Code de conduite

Ce projet applique le [Contributor Covenant](https://www.contributor-covenant.org/fr/version/2/1/code_of_conduct/),
version 2.1.

En bref : sois respectueux, bienveillant et constructif. Critique les idées et le code,
jamais les personnes. Aucun harcèlement, aucune discrimination.

Les comportements inacceptables peuvent être signalés de façon privée via l'onglet
[Security](https://github.com/daraook/vidai/security/advisories/new) du dépôt ou en
mentionnant le mainteneur (@daraook) dans une issue. Chaque signalement sera examiné.
